# 📊 Data Analytics Project

## Overview

This project demonstrates a complete end-to-end data analytics workflow, starting from raw data and ending with interactive dashboards and business insights. The project includes data loading, cleaning, exploratory data analysis (EDA), SQL analysis using MySQL, Power BI dashboard development.

The objective is to transform raw data into meaningful insights that support better decision-making.

The dataset contains structured business data used for analysis and visualization.

### Data Processing Includes:
- Loading the dataset using Python
- Handling missing values

- Data cleaning and preprocessing
- Feature exploration

## How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/data-analytics-project.git
cd data-analytics-project
```

### 2. Install Required Libraries

```bash
pip install -r requirements.txt
```

### 3. Run Python Scripts

```bash
python data_loading.py
python eda.py
python data_cleaning.py
```

### 4. Import Data into MySQL

- Create a database in MySQL.
- Import the cleaned dataset.
- Execute the SQL queries provided in the `SQL` folder.

### 5. Open the Power BI Dashboard

- Open the `.pbix` file in Power BI Desktop.
- Refresh the dataset if needed.
